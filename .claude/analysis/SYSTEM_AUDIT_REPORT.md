## 🚨 關鍵安全問題（必須立即處理）

### 🔴 CRITICAL（立即修復）

1. **敏感信息洩露** — `.env` 檔案包含實際 Google OAuth2 祕密
   - 影響：攻擊者可冒充 Google 帳號進行 OAuth2 劫持
   - 修復：立即重設 Google Client Secret、移除 `.env` 至 `.gitignore`
   - **修復時間**：1 小時

2. **異常捕獲過度寬泛** — `except Exception` 在多個關鍵位置
   - 位置：`auth.py:36`, `routers/auth.py:76`, `rate_limit.py:79`
   - 影響：掩蓋真實錯誤、無法偵測攻擊、Rate limit DoS 風險
   - **修復時間**：30 分鐘

3. **超級管理員權限風險** — 環境變數 `SUPERUSER_EMAILS` 生產未限制
   - 影響：任何具名 email 可自動獲得全權
   - 修復：移除或限制該變數、使用 IP 白名單或 2FA
   - **修復時間**：30 分鐘

4. **缺失 CSRF 保護** — POST/PATCH/DELETE 端點無 CSRF token 驗證
   - 影響：跨站請求可修改資源
   - **修復時間**：1 小時

### 🟠 HIGH（高優先級，1 週內解決）

5. **關鍵異常日誌缺失** — 郵件發送失敗只警告
   - 影響：無法追蹤業務流程失敗
   - **修復時間**：1 小時

6. **輸入驗證缺失** — 字號生成無長度驗證
   - **修復時間**：30 分鐘
